# HTML 研修教材

新入社員（HTML 初学者）向けの学習教材です。**上から順に進めれば一通り学べる**構成になっています。
困ったときは [`99_リファレンス_調べ方`](99_リファレンス_調べ方/) を引いてください。

## 学習の進め方

| 順番 | フォルダ | 学ぶこと | 目安 |
| --- | --- | --- | --- |
| 1 | [00_はじめに](00_はじめに/) | 環境構築（VSCode / 拡張機能 / Live Server） | 約 30 分 |
| 2 | [01_導入_Eラーニング](01_導入_Eラーニング/) | HTML / CSS の基礎（Udemy 教材） | 数日 |
| 3 | [02_自己紹介ページ](02_自己紹介ページ/) | 自分の自己紹介ページを作る | 1〜2 日 |
| 4 | [03_模写コーディング](03_模写コーディング/) | 既存サイトを模写して総合力をつける | 2〜3 日 |
| 5 | [04_CSSフォーム](04_CSSフォーム/) | フォームの見た目を CSS で作る | 1〜2 日 |
| 6 | [05_JSフォーム](05_JSフォーム/) | JavaScript でフォームを制御する | 1〜2 日 |
| 7 | [06_WebAPI](06_WebAPI/) | WebAPI と連携してデータを表示する | 1〜2 日 |
| 随時 | [99_リファレンス_調べ方](99_リファレンス_調べ方/) | 「困ったこと」から探せる逆引き集 | — |

> 「目安」はあくまで参考です。理解度に合わせて調整してください。

## まずやること

1. [資材をダウンロード](https://github.com/ITSVAPP/htmlkenshu/archive/refs/heads/main.zip)（または下記の git clone）して解凍する
2. [00_はじめに](00_はじめに/) を開いて環境構築をする
3. あとは上の表の順番に進める

### git を利用する場合

```
git clone https://github.com/ITSVAPP/htmlkenshu.git
```

- 解答を見たいとき: `git checkout answer` ／ 戻すとき: `git checkout main`
- まずは各課題の説明（目標・学習事項）を読み、自分で取り組んでから解答を確認しましょう。
