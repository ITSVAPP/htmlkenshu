## 進め方

`10_課題_入力要素` から順に進めます。各課題フォルダの `学習事項.md`（やること）と `参考.url`（解答デモ）を見ながら取り組みましょう。

1. 10_課題_入力要素 … input 要素の基本

7. 90_課題_総合問題 … 仕上げ

## 考え方・ヒント

- 見た目の出し分けは **CSS の状態セレクタ**が鍵: `:focus`（入力中）, `:valid` / `:invalid`（入力チェックの結果）, `:required`。
- 入力チェックは、まず HTML の属性（`required`, `type="email"`, `pattern`, `minlength`）でできることを使う。
- 困ったら [99_リファレンス_調べ方](../99_リファレンス_調べ方/) と [MDN: :invalid](https://developer.mozilla.org/ja/docs/Web/CSS/:invalid)。
