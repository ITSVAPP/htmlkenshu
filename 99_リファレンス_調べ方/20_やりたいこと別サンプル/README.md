# やりたいこと別サンプル（逆引き）

「**○○したい**」から、すぐ使える**サンプルコード**を探せます。

- サンプルは [`../サンプル/`](../サンプル/) にあります。**Live Server で開く**と動きを確認できます。
- ファイルを開くと、**コメントで 1 行ずつ説明**しています。値を書き換えて、変化を試してみましょう。
- 表の「**使うもの**」だけで分かるものは、ファイルを開かなくても OK です。

> ここに無いことは [30_調べ方とAIプロンプト](../30_調べ方とAIプロンプト/) を見て、自分で調べる練習をしましょう。
