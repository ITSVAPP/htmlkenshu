# VSCode ショートカット（初心者向け・優先度順）

操作が速くなると学習もはかどります。**上から順に**覚えれば OK。まずは「★まず覚える」だけで十分です。
（Windows 表記。Mac は `Ctrl`→`⌘`、`Alt`→`option` に読み替え）

## ★ まず覚える（これだけでも効果大）

| ショートカット          | できること                       | ひとこと                             |
| ----------------------- | -------------------------------- | ------------------------------------ |
| `Ctrl + S`              | 保存                             | Live Server は保存で自動更新される   |
| `Ctrl + Z` / `Ctrl + Y` | 元に戻す / やり直し              | 間違えても怖くない                   |
| `Ctrl + C` / `V` / `X`  | コピー / 貼り付け / 切り取り     | 何も選ばず実行すると「その行」が対象 |
| `Ctrl + /`              | コメントのオン・オフ             | コードを一時的に無効化できる         |
| `Alt + Shift + F`       | コード整形（自動できれいに）     | インデントがそろう                   |
| `!` → `Tab`             | HTML のひな形を一発入力（Emmet） | 新規 `.html` で                      |

## ◎ よく使う（次に覚える）

| ショートカット          | できること             | ひとこと                       |
| ----------------------- | ---------------------- | ------------------------------ |
| `Ctrl + F` / `Ctrl + H` | 検索 / 置換            | ファイル内をさがす・置きかえる |
| `Ctrl + D`              | 同じ単語を次々選択     | まとめて書き換えできる         |
| `Alt + ↑ / ↓`           | 行を上下に移動         | 並べ替えが速い                 |
| `Shift + Alt + ↑ / ↓`   | 行を上下に複製         | 似た行を増やす                 |
| `Ctrl + P`              | ファイルをすばやく開く | 名前を打つだけ                 |
| `Ctrl + Shift + P`      | コマンドパレット       | 機能名で何でも実行             |

## ○ 覚えると便利

| ショートカット     | できること                       |
| ------------------ | -------------------------------- |
| `` Ctrl + ` ``     | ターミナルの開閉                 |
| `Ctrl + B`         | サイドバー（ファイル一覧）の開閉 |
| `Ctrl + Shift + L` | 同じ単語を全部いっぺんに選択     |
| `F2`               | 変数名などの一括リネーム         |
| `Ctrl + Shift + K` | 行を削除                         |
| `Alt + クリック`   | カーソルを複数置く（同時編集）   |

## ブラウザ側（Live Server で確認中に）

| ショートカット | できること                             |
| -------------- | -------------------------------------- |
| `F5`           | 画面の更新                             |
| `F12`          | 開発者ツール（エラー確認・要素の調査） |

## すべてのショートカットを見たいとき

VSCode で `Ctrl + K` → `Ctrl + S` を押すと「キーボードショートカット一覧」が開きます。検索して自分用に変更もできます。
