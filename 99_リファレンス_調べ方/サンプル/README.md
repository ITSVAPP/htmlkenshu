# サンプル集

「やりたいこと」ごとに、**最小限のシンプルな HTML** を 1 ファイルにまとめています。
ファイルを Live Server で開くと動きを確認できます。コードを見て、まねして使ってください。

逆引き（困りごとから探す）は [99_リファレンス_調べ方 の README](../README.md) からどうぞ。

## ファイル一覧

### 基本（HTML）

- 見出しを付けたい.html
- 文字を太字にしたい.html
- 改行したい.html
- 箇条書きにしたい.html
- 画像を表示したい.html
- リンクを貼りたい.html
- 表を作りたい.html

### 見た目（CSS）

- 文字の色やサイズを変えたい.html
- 文字を中央に寄せたい.html
- 背景に色を付けたい.html
- 背景に画像を敷きたい.html
- classでスタイルを付けたい.html
- 要素を横に並べたい.html
- CSSを外部ファイルに分けたい.html

### 応用

- マウスを乗せたら変化させたい.html
- ファビコンを設定したい.html
- Googleマップを埋め込みたい.html

## 使っている素材

- `images/写真.jpg`, `images/背景.png` … 表示確認用の画像
- `images/favicon.ico` … ファビコンの例
