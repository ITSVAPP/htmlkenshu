# サンプル集

「やりたいこと」ごとに、**コメント付きの HTML** を 1 ファイルにまとめています。
Live Server で開くと動きを確認できます。**コードのコメントを読みながら**、まねして使ってください。

逆引き（困りごとから探す）は [`../20_やりたいこと別サンプル/`](../20_やりたいこと別サンプル/) からどうぞ。
